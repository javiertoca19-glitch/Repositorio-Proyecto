# Gastronomía Universal

Proyecto desarrollado para la evidencia AA5_EV03.

## Descripción
Sistema tipo restaurante digital que permite crear pedidos mediante una API REST.

## Arquitectura
- Models → Simulación de base de datos
- Services → Lógica de negocio
- Controllers → Manejo de peticiones
- Routes → Definición de endpoints

## Endpoint principal

POST /api/pedidos

## Tecnologías
- Node.js
- Express
