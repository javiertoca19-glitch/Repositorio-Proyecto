##  Descripción

Este proyecto consiste en el desarrollo de una API REST para la gestión de pedidos del restaurante digital **Gastronomía Universal**.

La API permite:
- Crear pedidos
- Consultar todos los pedidos

Se implementó utilizando arquitectura REST y separación de responsabilidades (Modelo, Servicio, Controlador y Rutas).
